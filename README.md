#  Plateforme de gestion des sujets de mémoire de licence

##  Description
Cette plateforme permet de gérer les sujets de mémoire de licence entre les étudiants, les encadrants et l'administration.

##  Objectifs
- Centraliser les sujets de mémoire
- Faciliter la communication
- Automatiser les candidatures

##  Acteurs
- Étudiant
- Encadrant
- Administrateur

##  Fonctionnalités

### Étudiant
- Consulter les sujets
- Postuler à un sujet
- Suivre candidature

### Encadrant
- Proposer un sujet
- Valider/refuser candidatures

### Administrateur
- Gérer les utilisateurs

## Architecture
- Frontend : java fx
- Backend : Java fx 
- Base de données : MySQL

## Base de données
Tables principales :
- Utilisateur
- Étudiant
- Encadrant
- Sujet
- Candidature
